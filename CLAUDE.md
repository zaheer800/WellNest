## Quick Reference

- **Stack:** React 18 + TypeScript + Tailwind CSS + Supabase + Claude API
- **Brand colours:** Navy `#0A2540` · Teal `#0EA5B7` · Green `#22C55E`
- **Font:** Poppins (SemiBold headings, Regular body)
- **Deploy:** Vercel (frontend) + Supabase (DB, Auth, Edge Functions)
- **Active branch:** `claude/plan-app-revamp-DIjLL`
