# WellNest Design Language (Final)

## Brand Essence
Health Records. Smarter Insights. Better Decisions.

## Colors
Primary: #0A2540 (Deep Trust)
Accent: #0EA5B7 (Clarity)
Growth: #22C55E (Progress)

## Icon System
- Dark background required
- 2 arc structure (support + system)
- Geometric heart (precision)
- Subtle AI nodes (optional)

## Spacing
Use 8px grid (8,16,24,32,48)

## Typography
Poppins:
- Headings: SemiBold
- Body: Regular

## UI Principles
- Minimal over decorative
- Data-first layout
- Clear hierarchy

## Buttons
Primary: Solid Teal
Secondary: Outline

## Logo Rules
- Always centered
- Maintain padding = 20%
- Do not distort or recolor

## Tone
- Clear
- Calm
- Intelligent
